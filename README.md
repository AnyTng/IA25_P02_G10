# IA25_P02_G10

Projeto de Inteligência Artificial com três notebooks que cobrem classificação, clustering e regras de associação usando dados do EA FC e Jogos Olímpicos.

- `notebook1.ipynb`: aprendizagem supervisionada para classificar jogadores do EA FC por grupo de posição (GK, DEF, MID, ATT) com atributos físicos e técnicos.
- `notebook2.ipynb`: clustering com KMeans para descobrir perfis de jogadores do EA FC 24 a partir de atributos detalhados.
- `notebook3.ipynb`: regras de associação Apriori para estimar a probabilidade de medalha num desporto sabendo que outra medalha foi conquistada noutro.
